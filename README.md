# Portfolio — Matthew Oluwanifemi

Frontend developer portfolio site and CV.

## Overview
This repository contains a simple portfolio site (HTML/CSS/JS) and a web-friendly CV (`cv.html`). The site is lightweight and focused on showcasing frontend projects, skills, and contact information.

## Files
- `portfolio_index.html` — main portfolio page
- `styles.css` — site styles
- `scripts.js` — portfolio interactive scripts
- `cv.html` — printable HTML CV
- `assets/` — supporting assets (images, PDFs)

## Local preview
Open the `portfolio_index.html` file in your browser to preview the site locally:

1. In File Explorer double-click `portfolio_index.html`, or
2. From PowerShell (in project root):

```powershell
# Open in default browser
start .\portfolio_index.html
```

## How to update
- Edit the HTML files in the project root.
- Styles are in `styles.css` and scripts in `scripts.js`.
- Update `cv.html` for CV changes; use your browser Print → Save as PDF to export.

## Deployment
You can host this site on GitHub Pages (Settings → Pages → branch `main` → root) or Netlify Drop.
